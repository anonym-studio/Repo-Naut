# Repo-Naut プロダクトLP 設計・仕様書

## 1. コンセプト

「Repo-Naut（リポ・ノート）」は、煩雑になりがちなローカルリポジトリの宇宙（ワークスペース）を自在に航行し、管理・整理するための「コックピット」です。

- **タグライン**: Your Repos. Your Mission. (あなたのリポジトリを、あなたのミッションに。)
    
- **キーワード**: 航行 (Navigate), 整理 (Organize), 加速 (Accelerate)
    

## 2. デザインガイドライン

ロゴの配色とシンボルを基調とした「宇宙・サイバー・モダン」なデザインを採用します。

### カラーパレット

- **Deep Space (Background)**: `#05070A` (漆黒に近いネイビー)
    
- **Nebula Blue (Primary)**: `#3B82F6` (鮮やかなブルー)
    
- **Cosmic Purple (Secondary)**: `#8B5CF6` (神秘的なパープル)
    
- **Starlight (Text)**: `#F8FAFC` (視認性の高いオフホワイト)
    

### フォント・タイポグラフィ

- **ロゴフォント風**: Sans-serif, Bold (未来的な印象)
    
- **本文**: Noto Sans JP, Inter
    

## 3. ページ構成 (セクション)

1. **Hero Section**: 宇宙飛行士ロゴを大きく配置。アプリの第一印象を決定づける。
    
2. **The Problem (Pain Point)**: 「プロジェクトが増えすぎて、どこに何があるか分からない」という課題を提示。
    
3. **Core Features (The Solution)**:
    
    - **Scanning**: 一瞬でリポジトリを発見。
        
    - **Mission Control**: 状態確認、GitHub連携、エディタ即起動。
        
4. - **Space Station (Kanban)**: リポジトリに紐づいたタスク管理。
        
5. **Technology**: Rust + Tauri による圧倒的な軽量さと高速性能の強調。
    
6. **Call to Action (CTA)**: インストール・ダウンロードへの誘導。
    

## 4. 訴求ポイント

- **「探す」時間を「作る」時間へ**: コマンドラインやFinderを彷徨う時間を削減。
    
- **リポジトリ中心のタスク管理**: 既存のタスク管理ツールとの差別化（リポジトリに密着）。
    
- **洗練されたUI**: 開発のモチベーションを上げる、ダークモードに最適化された美しさ。
    

## 5. ターゲット

- 同時に複数のプロジェクトを抱えるフリーランスエンジニア
    
- 過去のコード資産（リポジトリ）を整理したい研究熱心な開発者
    
- GitHub CLI (gh) を愛用し、さらに効率を上げたいパワーユーザー
