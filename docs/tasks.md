## 更新ルール

1. 新しいタスクは **「直近で着手したいもの」** の末尾に追加
2. 着手したら `🚧 (作業中)` を先頭に付ける
3. 完了したら `[x]` にする（行は消さない）
4. PR / コミットで関連するタスクには PR 番号やコミット SHA を末尾に追記する
   - 例: `- [x] 🟡 ... — #42 (2026-05-13)`
